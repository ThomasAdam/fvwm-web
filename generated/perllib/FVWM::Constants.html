NAME

FVWM::Constants - exports FVWM specific constants  

DESCRIPTION

This class is used to load FVWM constants into your perl script or class. Note, if you use FVWM::Module the constants are automatically exported. So you usually do not need to include this class.  

USAGE

  use FVWM::Constants;
 

SEE ALSO

For more information, see the FVWM::Module manpage.
