NAME

index - lists all available help topics  

DESCRIPTION

Recent fvwm versions install the Perl library that makes creating FVWM modules in Perl possible and easy.

You may read the Perl library documentation locally by running: 

    % fvwm-perllib man <topic>
Available topics: 

    index
    tutorial
    FVWM::Constants
    FVWM::Event
    FVWM::EventNames
    FVWM::Module
    FVWM::Module::Gtk
    FVWM::Module::Tk
    General::FileSystem
    General::Parse
For example: 

    % fvwm-perllib man FVWM::Module
